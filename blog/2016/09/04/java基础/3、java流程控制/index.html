六、if和switch比较

  1. 相同点：
    都可以处理多分枝结构，处理多种情况。
  2. 不同点
    switch：只能处理数据类型为：byte short char int String 的数据，且没种情况数据为固定值不可以为范围值。
    if: 特别适合处理范围值数据。

七、单重for循环结构

  1. 程序中往往需要重复执行某些代码或功能，为了方便做此类动作就需要用到循环。
  2. 循环可分为：for while do-while foreach
  3. 循环三要素
    • 循环条件
    • 循环操作
    • 条件变化

  4. for循环语法：

    其中条件初始化可以定义在外面，但是当此循环结束后初始化变量仍有效且保持循环结束后的值，条件判断可以控制循环执行次数，
    条件变化可以写在循环体语句中，但是不建议这样做。

  5. 执行流程：
    首先执行条件初始化，且只执行一次，然后进行条件判断，满足则执行循环体语句（不满足直接跳出执行后面代码），循环体语句执行结束后执行条件变化，第一次循环结束。
    接着再次进行条件判断，满足则执行循环体语句（不满足直接跳出执行后面代码），循环体语句执行结束后执行条件变化，第二次循环结束。
    程序依次按此流程执行剩余的循环。。。。。。
    当最后一次循环执行完后，执行条件变化，此时条件将不满足，但是程序会再次进行条件判断，不满足，循环结束。

  7. for循环细节：

    • for循环后的()里可以省略（条件初始化、条件判断、条件变化，但是不能省略 两个分号(; ;))。

    • for循环()里初始化的变量的作用域为这个for循环，出了for循环变量将无意义，所以循环外部无法获取在()里初始化的循环变量。

    • 缺少循环条件，或者循环条件永远满足，或者无条件改变造成死循环
