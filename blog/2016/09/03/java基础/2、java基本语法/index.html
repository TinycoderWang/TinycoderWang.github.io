Java基础（二）：基本语法

此系列文章是本人学习时所做笔记，现在做一个整理，大神请略过。。。

一、关键字

  1. 什么是关键字：
    被Java语言占用并赋予了特殊含义和作用的单词。
  2. 关键字的特点：
    • 全部为小写。
    • 被Java自身霸占，在自己定义变量、方法名、类名等时不能使用。

  3. 关键字的分类：

    • 用于定义数据类型的关键字:
      class（类） interface（接口）
      byte short int long float double char boolean（数据类型）
      void

    • 用于定义数据类型值的关键字
      true（真） false（假） null（空）

    • 用于定义流程控制的关键字
      if else （条件选择）
      switch case default（多重分支）
      while do while for（循环）
      break continue return （执行流程控制）

    • 用于定义访问权限修饰符的关键字
      private（私有） protected（保护） public（公有） （什么也不写：默认）

    • 用于定义类、方法、变量修饰符的关键字
      abstract（抽象） final（最终） static（静态） synchronized（同步）

    • 用于定义类与类、类与接口、接口与接口之间关系的关键字
      extends（继承） implements（实现）
      注意：类只能单继承，但是可以多实现；接口可以多继承

    • 用于定义建立实例及引用实例，判断实例的关键字
      new（创建） this（对象自身的引用） super（父类对象的引用） instanceof（类型判断）

    • 用于异常处理的关键字
      try（包含可能出现异常的代码块）
      catch（捕获到的异常）
      finally（最终要执行的代码，一般为释放资源操作）
      throw（抛出异常，需要new出来）
      throws（声明异常，必须由调用者处理）

    • 用于包的关键字
      package（包）
      import （导包）

二、标识符

  1. 什么是标识符：
    • 对各种变量、方法和类等命名时使用的字符序列，称为标识符。
  2. 标识符的命名规则：
    • 由字母、数字、_、$ 组成。
    • 不能以数字开头。
    • 严格区分大小写。
    • 标识符最好能使别人见名知意，如要定义一个人的类，则写 public class Person{}。

三、什么是变量：

  1. 在程序中，有一些量需要根据不同的条件进行值的改变，这些量称为变量。
  2. 程序是在计算机的内存中运行的，内存又被划分为很多的区域，每个区域都可以存储数据，
    计算机根据数据所在区域的地址（如0X0000FF）在不停的进行数据的存储、获取与改变。
    程序开发人员无法得知某个数据具体存储在哪个地址中，为了开发方便则出现了变量。（直观、方便使用）
  3. 变量就是程序员给某个内存地址所起的名字。

  • 浮点类型：float double

  • 布尔型：

boolean 占用1bit
  • 引用数据类型：
    • 字符串型：
      String 存储一串字符，如”Hello world”、”你好”等。
    • 对象等的引用
      Object obj = new Object();
  • 整数（byte short int long）
    整数是按照在内存中的大小进行划分的，其中最常用的为int。
  • 浮点数（float double）
    浮点数是按照精确度的高低来进行划分的，其中double的精确度高，默认的浮点数类型为double。
  • 字符型（char）
    表示单个字符，取值范围为0~65535，可以表示出全部的unicode编码表里的字符。
    在0-127范围内,char和byte/short/int/long之间可以互相转换（因为这些数据类型的取值在0-127重叠）
  • 字符串型（String）
    String其实为java.lang包里的一个类，且被final修饰不能被继承。
    字符串是常量；它们的值在创建之后不能更改。字符串缓冲区支持可变的字符串。因为 String 对象是不可变的，所以可以共享。（JDK说明文档）
  • 常量
    常量是长久不会改变的量，如 1 、 3.1425926 、Hello 、辉 等都属于常量。
    整数常量的默认数据类型为 int 如果常量的值超过了int范围则需要定义如下：long ln = 999999999999999L;
    浮点数常量的默认数据类型为 double 如果需要将浮点数常量定义为float类型则需要定义如下： float fl = 3.1415F;

七、赋值运算符（=）

  1. = 为赋值运算符，不是数学上的等于。
  2. = 可以跟算术运算符进行组合使用。如：+=、-+、*=、/=、%=
  3. 如果一行代码中有多种运算,那么 = 最后进行。

八、比较运算符（ > < == >= <= != ）

  • 比较运算符的结果为boolean型，只有true或false两种结果。

  • 注意位运算符：
    << 左移运算（相当于乘以2）
    >> 右移运算（相当于除以2）
    >>> 无符号右移（空位补零）

九、逻辑运算符(关系运算符)（ & && | || ! ^ ）

  1. 逻辑运算符结果为boolean型，只有true或false两种结果。
  2. & && 与 两边同true才为true，只要有一个false则为false。其中 && 左边为false则不需要判断右边。
  3. | || 或 两边同false才为false，只要有一个true则为true。其中 || 左边为true则不需要判断右边。
  4. ！ 非 取反操作。 ！ture == false。
  5. ^ 异或 异或跟或只有一个差别:前后都是true,结果反而是false。

十、条件运算(三元运算)

  1. 格式： (条件表达式) ? 表达式1 : 表达式2 ;
    其中条件表达式的值必须为boolean型，且条件运算一定有结果。

  2. 执行的流程：
    首先判断条件表达式的值，如果条件表达式值为true则运算结果为表达式1的值，如果条件表达式值为false则运算结果为表达式2的值。

  3. 运用： 条件运算一般用来求最值

十一、数据类型转换

  1. 在Java中,只能进行同类型的运算,同类型的两个数据运算后的结果也必须是同类型。

  2. 自动类型转换
    Java在进行数据运算时会进行数据类型自动转换：
    小范围会自动往大范围转换 低精度会自动往高精度转换
    在运算时自动类型转换为：byte,short,char –> int –> long –> float –> double

  3. 强制类型转换

    Java只会自动的将数据从小范围转换为大范围，低精度转换为高精度，不能反向转换，为了能反向转换，Java提供了强制类型转换。
