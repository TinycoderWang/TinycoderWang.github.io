Java基础（二）：基本语法

发表于 2016-09-03   |   分类于 Java基础   |     |   阅读次数

此系列文章是本人学习时所做笔记，现在做一个整理，大神请略过。。。

一、关键字

  1. 什么是关键字：
    被Java语言占用并赋予了特殊含义和作用的单词。
  2. 关键字的特点：
    • 全部为小写。
    • 被Java自身霸占，在自己定义变量、方法名、类名等时不能使用。

  3. 关键字的分类：

    • 用于定义数据类型的关键字:
      class（类） interface（接口）
      byte short int long float double char boolean（数据类型）
      void

    • 用于定义数据类型值的关键字
      true（真） false（假） null（空）

    • 用于定义流程控制的关键字
      if else （条件选择）
      switch case default（多重分支）
      while do while for（循环）
      break continue return （执行流程控制）

    • 用于定义访问权限修饰符的关键字
      private（私有） protected（保护） public（公有） （什么也不写：默认）

    • 用于定义类、方法、变量修饰符的关键字
      abstract（抽象） final（最终） static（静态） synchronized（同步）

    • 用于定义类与类、类与接口、接口与接口之间关系的关键字
      extends（继承） implements（实现）
      注意：类只能单继承，但是可以多实现；接口可以多继承

    • 用于定义建立实例及引用实例，判断实例的关键字
      new（创建） this（对象自身的引用） super（父类对象的引用） instanceof（类型判断）

    • 用于异常处理的关键字
      try（包含可能出现异常的代码块）
      catch（捕获到的异常）
      finally（最终要执行的代码，一般为释放资源操作）
      throw（抛出异常，需要new出来）
      throws（声明异常，必须由调用者处理）

    • 用于包的关键字
      package（包）
      import （导包）
